# React Native Template
React Native项目模板

## 集成功能
- UI：Ant Design
- 网络：Axios网络请求
- 路由：React-Navigation
- 存储：react-native-storage



## 启动项目
**安装依赖**

```shell
npm install
```

or

```shell
yarn install
```



**启动项目**

```shell
npm run android
```

or

```shell
yarn android
```
