# UUID React Native 
UUID移动端 React Native APP 项目

## 集成功能
- UI：Ant Design; BeeShell
- 网络：Axios网络请求
- 路由：React-Navigation
- 存储：react-native-storage
- 状态管理：React-Redux



## 启动项目
**安装依赖**

```shell
npm install
```

or

```shell
yarn install
```



**启动项目**

```shell
npm run android
```

or

```shell
yarn android
```
